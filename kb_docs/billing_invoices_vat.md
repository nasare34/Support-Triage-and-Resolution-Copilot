# AcmeDesk Billing: Invoices & VAT

## Invoice copies
- Invoices are available in **Settings → Billing → Invoices**.
- If you need an emailed copy, provide the invoice ID and billing email.

## VAT issues
Common causes:
- Billing address mismatch
- Missing/invalid VAT ID
- Country change after renewal

Resolution:
- Update billing address and VAT ID, then request a regenerated invoice.
